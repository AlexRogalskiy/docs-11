# Logic



[`all`](../ref/all.md)

: Returns `1b` if all items are non-zero. (Aggregate)

[`&` `and`](../ref/minimum.md)

: Logical AND. Also Minimum. (Atomic)

[`any`](../ref/any.md)

: Returns `1b` if any item is zero. (Aggregate)

[`not`](../ref/not.md)

: Returns `1b` where (items of) argument are zero. (Atomic)

[`null`](../ref/null.md)

: Is null. (Atomic)

[`|` `or`](../ref/maximum.md)

: Logical OR.  And maximum. (Atomic)
