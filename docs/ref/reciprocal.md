# `reciprocal`


Syntax: `reciprocal x`, `reciprocal[x]` (unary, atomic)

Returns the reciprocal of numeric `x`.
```q
q)reciprocal 0 0w 0n 3 10
0w 0 0n 0.3333333 0.1
```
