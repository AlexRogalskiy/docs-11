# `hcount`



_File size_

Syntax: `hcount x`, `hcount[x]`

Returns as a long integer the size in bytes of file `x`.

```q
q)hcount`:c:/q/test.txt
42j
```

On a compressed file returns the size of the original uncompressed file.
